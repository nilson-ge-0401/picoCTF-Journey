## Packet Analysis
1. Packet: A single unit of network communication
2. Packet Capture: A collection of packets taken from a network interface
3. Packet Analysis: Inspecting and studying a packet capture
### OSI Layer Model
1. Physical Layer: Electrical current, radio signal
2. Data Link Layer: MAC Address
3. Network Layer: IP Address, Routing
4. Transport Layer: Ports
5. Session Layer
6. Presentation Layer
7. Application Layer: Web Browser, Spotify, Email Clients
### Segmentation
1. Stream: Back and forth between devices, A 'conversation'
2. Wireshark Tools: Follow TCP Stream, Export Objects
